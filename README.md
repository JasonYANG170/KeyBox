## 功能
- ✅支持从浏览器导入网站密钥
- ✅支持 TOTP 2FA实时验证码绑定
- ✅支持蓝牙模拟输入HID 
- ✅支持 WIFI NTP  校时
- ✅支持设置设备6位密码
- ✅支持息屏
- ✅支持屏幕亮度调节。
